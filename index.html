Make your own atlas

This whole globe is a single, self-contained HTML file — no build step, no server. Copy it, change a few lines, and it's yours.

  1. 1Copy the full code with the button below (or grab it from the repo).
  2. 2Paste it into a new file named index.html.
  3. 3Find the EDIT HERE block near the top of the script — set your GOAL, your REPO_URL, and replace the PLACES list with your own { name, region, date, lat, lng } entries (mark your base with home:true).
  4. 4Open the file in any browser — done. To publish, push it to GitHub and enable Pages.
View repo ↗

Tip: set REPO_URL in the code so this button points at your own repository.
